[1] Enable Your Awesome Border :-

to enable Awesome Border Plugin you need to insert tow Classes [AwesomeBorder , (yourSymbol)]
-- Example : 
<div class="AwesomeBorder ab-circle"></div>

[2] What Symboles I have ??
you have 3 Symbols you can Enable one of them by their classes :- 

(1)
Symbol name :  Circle
Symbol Class : ab-circle

(2)
Symbol name :  Triangle
Symbol Class : ab-triangle

(3)
Symbol name :  Oblong
Symbol Class : ab-oblong

[3] Can I Set My border size , color .. etc ?? 

of course ... 
you have two choices :- 
1- just insert your classes and it will work by defult values 
2- set you custom values 

[4] How Can I set my custom values ?

you have some Attributes to set your custom values 

(1)
Attribute Name : data-border-position
Attribute Default Value : top 
Attribute options : you have 2 options to set you border position [top , bottom]

(2)
Attribute Name : data-border-size
Attribute Default Value : 25
Attribute options : just put your size as a number  
*Note : this attribute mean symbol size 

(3)
Attribute Name : data-border-color
Attribute Default Value : #E84C4C
Attribute options : just put your color by Color name , color Hex , rgb , rgba  .... As you like

--------------------------
there is two Attributes specifically to oblong symbol ONLY 

(1)
Attribute Name : data-border-height
Attribute Default Value : border-size * 1.5
Attribute options : just put your spacing as a number 

(2)
Attribute Name : data-border-spacing
Attribute Default Value : 10
Attribute options : just put your spacing as a number 
***this attribute to put spacing between oblong symbols 

***NOTE : When you Choose "oblong symbol" will follows that :

symbol size = data-border-size - data-border-spacing 

so you shoulden't set  "border size value" = "border spacing value"
